UART, Universal Asynchronous Receiver-Transmitter

## Feature
TX RX
Variable Baud Rate 波特率
Point-to-point
Data Framing: Starting bit+data bits+optional parity bit+stop bit(s)
Half-duplex

2. Data Framing
3. Asynchronous Transmission
4. Asynchronous Reception
